Netflix Content Analysis Dashboard

Project Overview

This project is an interactive Netflix Content Analysis Dashboard created using Tableau Public. The dashboard helps analyze Netflix movies and TV shows data using different visualizations and filters.

The project provides insights into:

* Movies vs TV Shows comparison
* Genre distribution
* Country-wise content analysis
* Ratings distribution
* Movie duration analysis
* Release year trends
